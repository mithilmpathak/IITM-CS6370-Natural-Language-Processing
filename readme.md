# CS6370: Natural Language Processing

This repository contains my implementations and exploratory work for the **CS6370 – Natural Language Processing** course at **IIT Madras**, undertaken during the **Spring Semester 2026**.

The repository primarily includes assignments, experiments, and supporting material related to fundamental NLP concepts such as tokenization, text preprocessing, and information retrieval.

---

## 📂 Repository Structure

- **ASSIGNMENT/1/**  
  Foundations of Search Engines: text preprocessing, tokenization, stemming, lemmatization, TF–IDF representation, and similarity-based retrieval.

- **CLASS_SLIDES/** *(gitignored)*  
  Lecture slides and course material provided during the course.

- **READING_MATERIAL/** *(gitignored)*  
  Additional reading material and research papers related to topics such as PCFGs, Bayesian methods, and language modelling.
